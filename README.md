# PHP Unit Converter

A modern, web-based unit conversion utility built as part of my BCA coursework. This project features a "Glassmorphism" UI aesthetic and handles multiple types of mathematical conversions.

## Features
- **Temperature Conversion**: Celsius to Fahrenheit.
- **Data Storage Conversion**: Megabytes (MB) to Gigabytes (GB).
- **Weight Conversion**: Kilograms (KG) to Grams.
- **Modern UI**: Semi-transparent glass effect with smooth fade-in animations.
- **Responsive Design**: Optimized for viewing on devices like the ThinkPad E14.

## Tech Stack
- **Backend**: PHP 8.2 (XAMPP)
- **Frontend**: HTML5, CSS3 (Flexbox & Glassmorphism)
- **Version Control**: Git & GitHub

## How to Run Locally
1. Clone the repository to your XAMPP `htdocs` folder.
2. Start the Apache module in the XAMPP Control Panel.
3. Open your browser and navigate to `localhost/unit-converter-php/index.php`.
